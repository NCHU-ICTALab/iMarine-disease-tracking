### 1. 台灣疾管署 — 國際旅遊疫情建議等級表 ✅ 真
- **內容**：台灣疾病管制署每日公布「哪個國家正在流行什麼傳染病、警戒到第幾級」。
- **為何可信**：政府官方公告，是本系統判斷風險的**主要依據**。
- **資料時間**：每日更新；本次抓取最新一筆為 **2026-06-30**（例：烏干達馬堡病毒、寮國小兒麻痺、巴基斯坦瘧疾），共約 2,400 多筆，入庫 2,018 筆。
- **來源網站**：疾管署開放資料平台 https://data.cdc.gov.tw/
- **實際下載端點**：
  `https://www.cdc.gov.tw/CountryEpidLevel/ExportCSV?type=0&fileName=TCDCTravelAlertAll.csv`
- **對應程式**：`app/sources/cdc_scraper.py`
- **實測欄位**：`effective`(生效日)、`alert_disease`(疾病)、`severity_level`(警戒等級)、`ISO3166`(國別碼)、`areaDesc`(地區)、`circle`(經緯度)、`web`(來源連結)。

### 2. 世界衛生組織 WHO — 疾病爆發新聞（Disease Outbreak News, DON）✅ 真
- **內容**：WHO 公布的全球重大疫情事件；用來補強疾管署資料，並標記「國際關注公共衛生緊急事件（PHEIC）」。
- **資料時間**：即時；本次抓取最新一筆為 **2026-06-25**（印度立百病毒 Nipah），另有 6/24 黃熱病等。
- **來源網站**：https://www.who.int/emergencies/disease-outbreak-news
- **實際 API 端點**（OData JSON）：
  `https://www.who.int/api/news/diseaseoutbreaknews`
- **對應程式**：`app/sources/who_scraper.py`
- **實測欄位**：`DonId`、`PublicationDate`(公布日)、`Title`(格式「疾病 - 國家」，如 `Nipah virus disease - India`)。

## 三、船舶 AIS：已接真實來源（aisstream.io）

### 5. 進港船 AIS ✅ 真（即時串流，有航跡歷史限制）
- **現況**：`AIS_PROVIDER=aisstream` 時，`AISStreamProvider` 連 `wss://stream.aisstream.io/v0/stream`，
  訂閱台灣周邊 bounding box，收集即時 `PositionReport` / `ShipStaticData`，用「港口鄰近 + 低船速」偵測靠港。
- **真的部分**：船的 MMSI、IMO、船名、目的地（Destination）、即時經緯度、船速皆為**真實 AIS 資料**。
- **限制（重要）**：aisstream 免費串流**沒有航跡歷史**，單次快照無法還原「這艘船 10 天前在孟買」。
  系統把每次觀測累積到 `data/ais_sightings.json`，靠港序列隨多次執行（或掛排程）逐步補齊；
  且台灣周邊 box 主要能看到抵達台灣港口的船，要看到**外國港**需擴大 box 並長期收集。
  → 因此改真 AIS 後，`prev_foreign_port` 與風險比對在初期會較稀疏。
- **檔案／程式**：`app/sources/ais_provider.py`（`AISStreamProvider`）、
  收集腳本 `scripts/collect_ais.py`、累積檔 `data/ais_sightings.json`。
- **可切回模擬**：設 `AIS_PROVIDER=mock` 讀 `data/mock_ais.json`（6 艘刻意佈局的模擬船），做可重現 demo。
- **調校參數**（`.env`）：`AIS_COLLECT_SECONDS`（收集秒數）、`AIS_PORT_RADIUS_KM`（靠港判定半徑）、
  `AIS_PORT_MAX_SOG`（過境船速門檻）、`AIS_BBOX`（訂閱範圍）。
- **其他可替換來源**：AISHub（需貢獻接收器）https://www.aishub.net/api ；
  MarineTraffic / VesselFinder（付費、覆蓋率最佳、含航跡歷史）。

## 四、「高風險」結果代表什麼？

範例（船為模擬、疫情為真實）：

| 船（模擬） | 上一站（模擬） | 比對到的疫情（真實） | 風險 |
|---|---|---|---|
| OCEANDUKE07 | 印度 → 新加坡 | 印度 Nipah 病毒（WHO 2026/6/25 通報） | 高 |
| MANILAEXP12 | 菲律賓 → 香港 | 香港流感（疾管署通報） | 高 |

意義：**假設真有一艘船這樣航行，系統會拿真實疫情資料算出此警報。** 疫情端已是真實資料，
因此只要把模擬船換成真實 AIS，系統即可對真船發出真警報，**分析引擎無需改動**。
